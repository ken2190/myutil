utilmy.docs.zold_cli_format module

A simple python module to parse the code and format it based on some rules. Some rules are: rule 1 - change a line starting with 3 #’s into x #’s where x is 90 by default

if no text was found else preserve text and fill the rest with #’s

rule 2 - normalize log statements in the file rule 3 - put all consecutive imports on one line rule 4 - align assignment operators

Usage:

cli_format -i /path/to/file or /path/to/dir –dirout /path/to/output

Improved dictionary access through dot notation with additional tools.

Parameters

  • default_box – Similar to defaultdict, return a default value

  • default_box_attr – Specify the default replacement. WARNING: If this is not the default ‘Box’, it will not be recursive

  • default_box_none_transform – When using default_box, treat keys with none values as absent. True by default

  • default_box_create_on_get – On lookup of a key that doesn’t exist, create it if missing

  • frozen_box – After creation, the box cannot be modified

  • camel_killer_box – Convert CamelCase to snake_case

  • conversion_box – Check for near matching keys as attributes

  • modify_tuples_box – Recreate incoming tuples with dicts into Boxes

  • box_safe_prefix – Conversion box prefix for unsafe attributes

  • box_duplicates – “ignore”, “error” or “warn” when duplicates exists in a conversion_box

  • box_intact_types – tuple of types to ignore converting

  • box_recast – cast certain keys to a specified type

  • box_dots – access nested Boxes by period separated keys in string

  • box_class – change what type of class sub-boxes will be created as

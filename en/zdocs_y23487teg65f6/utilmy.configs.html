Return full path to the user-specific state dir for this application.
‚Äúappname‚Äù is the name of application.

If None, just the system directory is returned.

‚Äúappauthor‚Äù (only used on Windows) is the name of the

appauthor or distributing body for this application. Typically it is the owning company name. This falls back to appname. You may pass False to disable it.

‚Äúversion‚Äù is an optional version path element to append to the

path. You might want to use this if you want multiple versions of your app to be able to run independently. If used, this would typically be ‚Äú<major>.<minor>‚Äù. Only applied when appname is present.

‚Äúroaming‚Äù (boolean, default False) can be set True to use the Windows

Typical user state directories are:
